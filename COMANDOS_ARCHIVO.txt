# Comandos para configurar el sistema de carga de archivos

# 1. Ejecutar la migración para agregar columnas de archivo
php artisan migrate

# 2. Crear el enlace simbólico de storage (si no existe)
php artisan storage:link

# 3. Verificar que el directorio existe y tiene permisos correctos
# En Windows (PowerShell):
# New-Item -ItemType Directory -Force -Path storage\app\public\convocatorias

# 4. Si necesitas revertir la migración:
# php artisan migrate:rollback --step=1
